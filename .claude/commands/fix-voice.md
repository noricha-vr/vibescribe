# /fix-voice コマンド

## サブコマンド

### /fix-voice（引数なし）
fix-voiceスキルを実行。音声認識の誤変換を修正するため、ユーザーから「誤った状態」と「正しい状態」を確認し、適切な登録先に追加。

### /fix-voice analyze
履歴データを分析し、未修正の誤変換パターンを検出。

実行手順:
1. `uv run python scripts/analyze_history.py --days 7` を実行
2. 結果をユーザーに表示
3. ★マーク（キーワード共起あり）のパターンは同音異義語テーブルへの追加を提案
4. ユーザーが承認したら postprocessor.py の同音異義語テーブルを更新

## 引数の判定

- 引数が「analyze」の場合 → 分析モード
- それ以外 → fix-voiceスキルを実行
